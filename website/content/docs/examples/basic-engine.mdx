title: Basic engine
description: Stress the fixed-concurrency core with synthetic retries, failures, and result snapshots.
---

This Phase 2 artifact sends 1,000 synthetic requests through `BasicKind` at fixed concurrency 32, introducing a one-time retry for indices divisible by seven and permanent failures for indices divisible by 97. It consumes all 1,000 terminal result snapshots concurrently and verifies 11 terminal failures, a matching failed-handler count, and nonzero retries.
